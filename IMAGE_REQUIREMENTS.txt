ENCHANTED QUILL — GAMIFICATION IMAGE REQUIREMENTS
==================================================
Generated: 2026-04-05
Branch: feature/gamification-system

All images should match the app's dark fantasy / arcane aesthetic:
rich purples, deep navies, golden ambers, and mystical flourishes.
Recommended format: .webp (with .png fallback if needed).

==================================================
SECTION 1: PROFILE BANNERS (stored in: public/banners/)
==================================================
Banners are full-width profile header images (recommended: 1200×300px).
They are referenced via users.active_banner_path and can be awarded as
achievements of type "banner".

HOW TO ADD IMAGES TO THE DATABASE (SEED EXAMPLE)
==================================================

After adding image files, seed the Achievement records like this:

Achievement::create([
    'name'              => 'Silver-Tipped Quill',
    'slug'              => 'silver-tipped-quill',
    'description'       => 'Awarded for publishing 10 chapters.',
    'type'              => 'badge',
    'icon_path'         => 'badges/silver-tipped-quill.webp',
    'requirement_type'  => 'chapters_published',
    'requirement_value' => 10,
    'is_hidden'         => false,
]);

Then seed Vocation records:

Vocation::create([
    'name'                   => 'The High Scribe',
    'slug'                   => 'high-scribe',
    'description'            => 'A master of the written word.',
    'bonus_description'      => '+10% Ink (XP) gain from publishing.',
    'bonus_type'             => 'ink_multiplier',
    'bonus_value'            => '1.1',
    'required_achievement_id'=> Achievement::where('slug', 'silver-tipped-quill')->value('id'),
]);
